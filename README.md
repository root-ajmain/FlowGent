# FlowGent — Digital Ecosystems for Modern Brands

A cutting-edge digital agency website built with **React 19**, **Vite**, **TypeScript**, and **Tailwind CSS 4**.

## Tech Stack

- **React 19** + **Vite 6** — Fast builds, HMR, code splitting
- **TypeScript** — Full type safety
- **Tailwind CSS 4** — Utility-first styling
- **React Router** — Client-side routing with lazy-loaded pages
- **Motion (Framer Motion)** — Smooth animations
- **react-helmet-async** — SEO meta tags per page

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server on port 3000 |
| `npm run build` | Type-check + production build |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
